Edge AI Infastructure

Catherine Ordun, November 2019

Find Out More

Challenges

Data generated by edge devices and sensors require special consideration for networking and data pipelines. It is unclear which of the multiple frameworks for processing data streams (queues) in real-time is optimal. The hardware requirements remain a challenge for processing AI data from the edge devices and sensors. Integration with existing legacy systems or the cloud- remains a worthy challenge.

Objectives

Edge AI will review frameworks suitable for edge compute and stream processing. We will define the architecture requirements for an edge server, as well as ancillary edge devices. There will be a thorough review of both software and hardware options.

Progression
